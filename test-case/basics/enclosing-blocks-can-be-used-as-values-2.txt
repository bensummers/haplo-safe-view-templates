
Enclosing blocks can be used as values for other functions

---

<div>
    with(b) {
        if(^{a}) {
            "TRUE"
        } else {
            "FALSE"
        }
    }
</div>

---
{
    "a": true,
    "b": { }
}
---
<div>TRUE</div>

---
{
    "a": false,
    "b": { }
}
---
<div>FALSE</div>
